# Feddy iOS SDK

Feedback infrastructure for iOS apps. Part of [Feddy](https://feddy.app).

- **Language:** Swift 5.9+ / SwiftUI-first
- **Minimum target:** iOS 15
- **Distribution:** Swift Package Manager
- **License:** MIT
